## Instrucciones

1. Crea y despliega tu servidor, utiliza las librerias de express, nodemon y dotenv.
2. Versiona tu aplicacion en la version 1
3. Crea tu API utilizando el modelo MVC, teniendo los archivos de rutas, controladores y servicios.
4. Crea las rutas que consideres necesarias para manejar el CRUD de las tareas
5. Utiliza express.Router() para manejar las rutas en tu archivo de rutas. 
6. Utiliza los diferentes verbos y maneras de hacer las peticiones para poder hacer los siguientes verbos 
    1. GET
    2. POST
    3. PUT
    4. DELETE
7. Subelo a un repositorio en github, y entregalo en el class-center
